
## Break and Continue 

+ `break` dùng để thoát khỏi vòng lặp ngay lặp tức
+ `continue` bỏ qua lần lặp hiện tại và chuyển tới lần lặp tiếp theo
+ `loop`, `while`, `for`


Ví dụ `Break`:

```rust
for i in 1..10 {
    if i == 5 {
        break;
    }
    println!("{}", i);
}
```

Ví dụ `Continue`:

```rust
for i in 1..10 {
    if i % 2 == 0 {
        continue;
    }
    println!("{}", i);
}
```
