## For

+ Lặp qua các phần tử của một chuỗi , mảng, ... 


Ví dụ:

```rust
let items = [1, 2, 3, 4, 5];
for item in items {
    println!("{}", item);
}
```

+ Giới thiệu cách sử dụng `iter()`, `into_iter()`, `iter_mut()`

```rust 
let items = [1, 2, 3, 4, 5];
for item in items.iter() {
    println!("{}", item);
}
```
