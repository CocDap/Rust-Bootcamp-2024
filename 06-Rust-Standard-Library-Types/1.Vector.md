## Vector

### Định nghĩa 
+ Kiểu dữ liệu động, có thể mở rộng phần tử 
+ Các phần tử phải cùng kiểu dữ liệu 

### Khởi tạo

+ Khởi tạo vector rỗng 

```rust
    let v = Vec::new();
```

+ Khởi tạo vector có giá trị ban đầu

```rust
    let v = vec![1,2,3];
```

### Thêm phần tử
+ Thêm `mut`
+ Sử dụng method `push`

```rust
let mut v = Vec::new();
v.push(5);
v.push(6);
v.push(7);
```
### Đọc phần tử
+ Truy cập sử dụng chỉ số 

```rust
    let value = v[2];
```
+ Sử dụng phương thức `get`

```rust
    let value = v.get(2);
```

### Duyệt các phần tử 
+ sử dụng `for`

```rust
    let v = vec![1,2,3];
    for value in v {
        println!("{}", i);
    }
```

### Sửa đổi phần tử 
+ Thêm `&mut`
```rust
    let v = vec![1,2,3];
    for value in &mut v {
        *i +=1;
    }
```
